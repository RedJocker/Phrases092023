Description 

To see the phrases that you have added to the application, you need to print them somewhere on the screen. In this stage, you need to set up the RecyclerView so that it shows the list of the phrases on your activity screen.

Objectives

  1. Create an XML layout for your RecyclerView items. The view should contain a TextView with the phrase (ID: phraseTextView) and a Delete TextView to remove it (ID: deleteTextView). You are free to design the app however you want, but don't change the IDs of the two Views;
  2. Create an adapter for your RecyclerView. The adapter should show the list with your predefined phrases. Set an OnClickListener for the Delete TextView that will remove the item. In this stage, we don't need you to save user's actions in any way, so it's okay if you restart the app, but the deleted phrases are back;
  3. Show at least three phrases on the screen.
